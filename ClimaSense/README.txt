Developed with the help of Oday Suleiman in 2025

The application has a dynamic background and displays the weather information any location entered by the user.

The application gives a breakdown of the daily weather, weather for the next 5 days and the break down for the weather for one of those days as well.

Each element is displayed with a certain UI element for better readability

The design is made customizable by using CSS files to give the user more control over how the app looks.
